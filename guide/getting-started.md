# 시작하기

이 가이드는 프로젝트를 빠르게 시작하는 방법을 설명합니다.

## 사전 요구사항

* Node.js 18 이상
* npm 또는 yarn
* Git

## 빠른 시작

```bash
git clone https://github.com/example/sample.git
cd sample
npm install
npm run dev
```

{% hint style="success" %}
설치가 완료되면 `http://localhost:3000`에서 확인할 수 있습니다.
{% endhint %}
